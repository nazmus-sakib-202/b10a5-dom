Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a way for programs to work with web pages. It shows the layout of a page as a tree made of different parts. This lets programming languages like JavaScript change and control the content, layout, and style of a web page.

How do you select an element from the DOM?

To select an element from the DOM, we can use methods like `document.getElementById('myId')` to find an element by its ID, `document.querySelector('.myClass')` to select an element using a class, or `document.querySelectorAll('.myClass')` to get all elements with that class. These methods let us work with and change the elements on the page. Each method is useful for different situations, like getting one element or multiple ones.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a method where we add one event listener to a parent element instead of adding it to each child element. This makes our code run faster and easier to manage, especially when we add new elements later. With this approach, we can handle events for many child elements using just one function.

How do you manipulate an element's attributes and styles using DOM?

We can change an element's attributes using methods like `getAttribute()` to get a value, `setAttribute()` to set a new value, and `removeAttribute()` to remove an attribute. To change styles, we can use the `style` property or the `classList` methods to add or remove classes. This lets us make changes to both the attributes and the look of DOM elements easily.
